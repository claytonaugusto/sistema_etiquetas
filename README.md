## 🇧🇷 Português

### 📝 Sobre o Projeto
Este projeto nasceu da necessidade de organizar a rotina de cuidados com a saúde. O sistema permite a emissão de etiquetas claras e objetivas que informam ao usuário a **quantidade exata** e o **horário** em que cada medicamento deve ser administrado, evitando erros de dosagem e esquecimentos.

### ✨ Funcionalidades
* **Interface Intuitiva:** Cadastro rápido de medicamentos e horários.
* **Cálculo de Ciclos:** Definição de intervalos (ex: de 8 em 8 horas).
* **Layout Adaptado:** Etiquetas formatadas para fácil leitura (ideal para idosos).
* **Exportação:** Geração de arquivo pronto para impressão.
